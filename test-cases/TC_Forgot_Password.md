# TC - Recuperação de Senha

## Objetivo
Verificar se o usuário consegue solicitar a recupeeração de senha ao esquecer suas credenciais.

## Pré-condição
- Usuário possui caadastro ativo no sistema
- Usuário está na tela de login

## Passos
1. Acessar o site
2. Clicar na opção "Esqueci minha senha"
3. Informar um e-mail válido cadastrado
4. Clicar no botão "Enviar"

## Resultado esperado
- Sistema exibe mensagem informando que as instruções foram enviadaspara o e-mail
- Usuário não tem acesso ao sistema sem redefinir a senha
