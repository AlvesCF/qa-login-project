# Casos de Teste - Login

## Login com credenciais válidas
**pré-condição: Usuário cadastrado**
**Passos:**
1. Acessar o site;
2. Preecher username válido;
3. Preencher senha válida;
4. Clicar em "Entrar".

*Resultado espeado:*
Usuário é redirecionado para a página inicial.

## Login com carregamento infinito
**Pré-condição: Usuário cadastrado**
**Passos:**
1. Acessar o site;
2. Preencher username váiido;
3. Preencher senha válida;
4. Clicar em "ENTRAR".

***Resultado esperado:***
Sistema não deve ficar em carregamento infinito
