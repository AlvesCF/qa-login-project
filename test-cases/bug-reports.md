# Erro no redirecionamento ao clicar em "ENTRAR"

## Passos para reproduzir
1. Acessar o site
2. Preencher o campo de username;
3. Preencher o campo de senha;
4. Clicar no botão "ENTRAR".

## Resultado obtido
Ao execultar os passos, o sistema não carrega e não redireciona para a página inicial. 

## Resultado esperado
Ao clicar em "ENTRAR", o sistema deve redirecionar para a página inicial. 

## Ambiente
- Navegador: Chrome
- Dispositivo: Desktop
- Sistema operacional: windows 10
- Ambiente: Teste

## Evidências
- Tela permanece em carregamento inifinito após clique em "ENTRAR"
