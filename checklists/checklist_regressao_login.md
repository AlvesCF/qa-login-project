# Checklist de Regressão - Login

## Login
- [ ] Página de login carrega corretamente
- [ ] Campo de username aceita texto
- [ ] Campo de senha mascara os caracteres
- [ ] Botão "Entrar" está habilitado
- [ ] Login com credenciais válidas redireciona para a página inicial
- [ ] Sistema não fica em carregamento infinito
- [ ] Mensagem do erro aparece para credenciais inválidas
- [ ] Mensagem de erro é genérica (não indica qual campo está errado)
- [ ] Logout funciona corretamente

## Geral
- [ ] Não há erros visuais críticos
- [ ] Textos e mensagens são claros
- [ ] Sistema responde dentro de tempo aceitável
