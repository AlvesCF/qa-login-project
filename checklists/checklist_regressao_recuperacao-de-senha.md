# Checklist de Regressão - Recuperação de Senha

## Login
- [ ] Link "Esqueci minha senha" está visível na tela de login
- [ ] Redirecionamento para a página de recuperação funciona
- [ ] Campo de e-mail aceita formato válido
- [ ] Campo de e-mail não aceita envio vazio
- [ ] Mensagem de confirmação é exibida após envio
- [ ] Sistema não informa se o e-mail está cadastrado
- [ ] Usuário não acessa o sistema sem redefinir a senha

## Geral
- [ ] Não há erros visuais críticos
- [ ] Textos e mensagens são claros
- [ ] Sistema responde dentro de tempo aceitável
